# 问题整理

- relay\_log日志损坏，无法同步  
Last_Error: Error initializing relay log position: I/O error reading the header from the binary log  
	- 处理方法：  
	```
	没有GTID
	stop slave;
	change master to
	master_log_file='Relay_Master_Log_File',
	master_log_pos=Exec_Master_Log_Pos,
	master_auto_position=0;
	start slave;
	
	使用GTID  
	stop slave;
	reset slave;
	start slave;
	
	```
